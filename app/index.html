Como usar o HookPad

Estrutura do script

Escreva uma função def main() com os parâmetros que desejar. O HookPad detecta automaticamente os tipos e gera os campos de teste e o exemplo curl.

def main(
    nome: str,
    numero: int = 42,
    dados: dict = {},
    arquivo: bytes = bytes(0),
):
    print(f"Olá {nome}, número: {numero}")
    return {"ok": True, "nome": nome}
Tipos suportados
str Texto simples "valor"
int / float Número 42
bool Booleano true / false
dict JSON object {"chave": "valor"}
list JSON array [1, 2, 3]
bytes Arquivo binário base64 ou multipart
Retorno

O valor retornado por return vira o body da resposta HTTP:

return {"status": "ok"}   # → JSON
return b"\x89PNG..."      # → application/octet-stream (binário)
print("qualquer coisa")   # → vai no stdout da resposta
Enviar arquivo binário
# Via multipart (recomendado para arquivos)
curl -X POST "URL" \
  -H "X-Token: TOKEN" \
  -F "arquivo=@meu.pdf"

# Via JSON com base64
curl -X POST "URL" \
  -H "X-Token: TOKEN" \
  -H "Content-Type: application/json" \
  -d "{\"arquivo\": \"$(base64 -w0 meu.pdf)\"}"

# Body binário puro (injeta como param \"file\")
curl -X POST "URL" \
  -H "X-Token: TOKEN" \
  --data-binary @meu.pdf
Dependências

Apenas dê import no topo do script. O HookPad instala automaticamente ao salvar. Cada script tem seu próprio ambiente virtual isolado.
