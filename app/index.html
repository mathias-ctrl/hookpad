Estrutura do script
Escreva uma função def main() com os parâmetros. O HookPad detecta os tipos automaticamente.
def main(
    nome: str,
    numero: int = 42,
    dados: dict = {},
    arquivo: bytes = bytes(0),
):
    return {"ok": True, "nome": nome}
Tipos suportados
strTexto simples → "valor"
int / floatNúmero → 42
boolBooleano → true / false
dictJSON object → {}
listJSON array → []
bytesBinário → base64 ou multipart
Enviar arquivo binário
# Via multipart
curl -X POST "URL" -H "X-Token: TOKEN" -F "arquivo=@meu.pdf"

# Via JSON com base64
curl -X POST "URL" -H "X-Token: TOKEN" \
  -d "{\"arquivo\": \"$(base64 -w0 meu.pdf)\"}"
Retorno
return {"status": "ok"}   # → JSON direto
return b"\x89PNG..."      # → application/octet-stream
